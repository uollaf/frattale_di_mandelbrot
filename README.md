# frattale_di_mandelbrot
Il progetto unisce matematica e programmazione C++. Ogni punto dello schermo viene associato a un numero complesso e calcolato a ripetizione. Se il risultato rimane stabile, il punto entra nel frattale di Mandelbrot; se cresce troppo, viene escluso. Il codice colora i pixel in base a questo calcolo, usando SFML solo per la grafica.
